180/**
181 * @namespace IH
182 * @brief "auto" type namespace
183 *
184 * "auto" type sizing macros, uses src/config.h as input
185 *
186 * The idea behind this namespace is to let users set config.h items to
187 * whatever they want, sizing variables for the least amount of space, automatically.
188 * The preprocessor evaluates the if macros in a way that will leave typedefs that
189 * size type aliases to the minimum byte-width required automatically.  It also warns users
190 * about the change.  The only hand-holding that happens is for type sizing, it's up
191 * to you to make sure you have the resources for your settings.
192 * A very convenient feature for users.
193 */
