    class UserInput¶
    inline UserInput::UserInput(char *output_buffer = NULL, uint16_t *output_buffer_string_pos = NULL, uint16_t output_buffer_len = 0, const char *username = NULL, const char *end_of_line_characters = _default_end_of_line_characters, const char *token_delimiter = _default_token_delimiter, const char *c_string_delimiter = _default_c_string_delimiter)¶

    UserInput Constructor with NO output by default.

    Creates a new instance. If you want output, declare a buffer size, buffer, and index.

    Parameters

    • output_buffer – NULL

    • output_buffer_string_pos – NULL

    • output_buffer_len – ZERO

    • username – NULL

    • end_of_line_characters – EOL term, default is ‘\r\n’

    • token_delimiter – token demarcation

    • c_string_delimiter – c-string demarcation
