UserInput constructor, no output by default.

UserInput has no output by default due to the default values passed in by the constructor. The constructor disables output by setting _output_enabled_ to false if output_buffer is NULL.

Parameters

output_buffer_lendefault ZERO, size this to length of output_buffer

◆ addCommand()

void UserInput::addCommand ( CommandConstructor &  command)

adds user commands to the input process

This function inspects Parameters for errors and reports the errors to the user if they have enabled output. If an error is detected, in the root command or any of its subcommands the entire command tree is rejected, and no sizing for dynamically allocated variables takes place.
